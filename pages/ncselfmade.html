Installation

Nextcloud is one of the easiest web apps to install...

To install nextcloud, you are going to need some services...

Services like Apache or Nginx let us make our own web server.


Software requirements

You will need PHP 8.x, a Database (MariaDB/MySQL) and a web server like Apache2 or Nginx.

Optional: redis(recommended)

Hardware requirements

You need a good hardware to host a server like this... But I'm gonna be honest.

When I started with my own hombelab, I did it with an old laptop with an Intel i7-4700U, 6GB RAM DDR3, 512GB HDD.

You can use any trash you have at home to host nextcloud. But I recommmend you to use something better. Maybe with 8GB RAM DDR4 and SSD storage
